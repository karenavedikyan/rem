# RemCard Route Map (Этап 1)

Обновлено: 2026-03-04

## 1) Публичные primary-разделы (верхнее меню)

- `/` — Главная
- `/navigator/` — Навигатор
- `/request/` — Заявка
- `/catalog/` — Каталог
- `/partners/` — Партнёрам

## 2) Secondary-разделы (футер)

- `/about/` — О проекте
- `/contacts/` — Контакты и обратная связь
- `/docs/` — Документы
- `/privacy/` — Политика конфиденциальности
- `/terms/` — Пользовательское соглашение

## 3) Каталог и воронка клиента

- `/catalog/` — список позиций (товары/услуги)
- `/catalog/service/?id=:serviceId` — карточка позиции
- Переходы в заявку:
  - `/request/?serviceId=...&serviceTitle=...&serviceStage=...&serviceTaskType=...#request`
  - `/request/?promoId=...&promoTitle=...&promoPartner=...&promoBenefit=...#request`

## 4) Партнёрский сценарий

- `/partners/` — главная страница для партнёров
- `/partners/add/` — форма добавления партнёра (операционный сценарий)
- `/partner/cabinet/` — кабинет партнёра (MVP)
- `/partners-stores/` — витрина магазинов-партнёров (вынесена из primary-меню)

## 5) Клиентский кабинет (архитектурная заготовка)

- `/cabinet/` — кабинет клиента (этап 1, заглушка структуры)
  - Профиль
  - Заказы
  - Бонусы
  - Избранное
  - Настройки

## 6) Примечание по архитектуре

Primary-навигация построена по сценарию клиента:

1. понять этап → `/navigator/`
2. выбрать решения → `/catalog/`
3. оставить заявку → `/request/`

Вторичные и справочные сущности перенесены в footer или отдельные контекстные маршруты.
