# Лендинг — набор дизайнеров карточек маркетплейсов

Одностраничный сайт для набора 4 дизайнеров карточек товаров для Wildberries, Ozon и других маркетплейсов.

## Структура

- `index.html` — главная страница (hero, о проекте, требования, форма)
- `styles.css` — тёмная минималистичная стилистика
- `script.js` — форма, плавный скролл

## Форма

- **Без настройки:** при отправке открывается WhatsApp с предзаполненным сообщением — кандидат может отправить заявку вручную.
- **С Telegram-ботом:** в `script.js` укажите `BOT_TOKEN` и `CHAT_ID` — заявки будут приходить в Telegram автоматически.

## Запуск

Откройте `index.html` в браузере или разместите на любом хостинге (GitHub Pages, Vercel, Netlify).
